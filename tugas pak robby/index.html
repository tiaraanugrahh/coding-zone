Belajar JavaScript

1. Variabel

Variabel digunakan untuk menyimpan data. Deklarasi dengan let dan const.

// Contoh
let nama = "Budi";
const umur = 25;
console.log(nama, umur);

2. Fungsi

Fungsi adalah blok kode yang dapat dipanggil berkali-kali.

// Contoh fungsi
function sapa(user) {
  alert("Halo, " + user + "!");
}
sapa("Andi");

3. Pengkondisian

Gunakan if untuk mengambil keputusan.

// Contoh
let nilai = 80;
if (nilai >= 75) {
  console.log("Lulus");
} else {
  console.log("Tidak Lulus");
}

4. Looping

Loop for digunakan untuk mengulang.

// Contoh
for (let i=1; i<=10; i++) {
  console.log(i);
}

5. Array

Array digunakan untuk menyimpan banyak data.

// Contoh array
let angka = [1, 2, 3, 4, 5];
console.log(angka[0]); // 1

6. Object

Object menyimpan data dalam bentuk key-value.

// Contoh object
let orang = {
  nama: "Rina",
  umur: 22,
  hobi: ["membaca", "musik"]
};
console.log(orang.nama);
