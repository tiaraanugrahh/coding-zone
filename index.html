Pengertian HTML

html (hypertext markup language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. html berfungsi untuk menentukan struktur dan isi dari sebuah halaman, seperti teks, gambar, tautan, tabel, hingga form. setiap elemen pada html ditulis dalam bentuk tag yang berpasangan, seperti <p> untuk paragraf dan <h1> untuk judul.

seiring berkembangnya teknologi web, html kini sudah sampai versi 5 (html5) yang membawa banyak fitur baru seperti dukungan multimedia (audio dan video), elemen semantik, dan peningkatan aksesibilitas. html5 memudahkan pengembang untuk membuat halaman yang lebih interaktif tanpa harus banyak bergantung pada plugin eksternal seperti flash.

Fungsi HTML

  • Membentuk struktur dasar halaman web.
  • Menghubungkan antarhalaman web melalui hyperlink
  • Menyediakan semantik pada struktur konten
  • Menyematkan berbagai elemen seperti teks,gambar,video, dan tautan
  • Mendukung kerja sama dengan CSS dan JavaScript untuk tampilan dan interaksi

Contoh Kode HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Contoh Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
  </body>
</html>
